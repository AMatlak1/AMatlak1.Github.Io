Java 2D Game Project Cyborgs vs Humans

My Java 2D Game project is a game called Cyborgs vs Humans
this game consists of the player, being the cyborg which has to protect
himself from the human's that are out to kill his kind. The game consists of 3
levels which gradually get harder, as the player move's through the level's more
complicated platform's are added to the game such as moving platform or harder
enemy's to kill, also the player must be careful to not come into contact with
some of the enemy's such as the Rock Human or the Cap Human, if the player comes
into contact with these enemy's they will be transported straight to level one
and die.

This 2D Java Game project allowed me the opportunity to learn more about
GUI's, working with files and also it allowed me to to learn more about
object oriented programming. Also another lesson i learned from this is about
the game user interface and how the characters interact with collision from enemy's
and objects.
